Descripción detallada

Clase abstracta que representa un modelo que tiene vistas.

Documentación del constructor y destructor

Modelo::~Modelo (  )  [pure virtual]

Destructor por defecto. Cuando el modelo se destruye, avisa a todas las vistas que tuviera para que dejen de apuntar a este modelo.

Documentación de las funciones miembro

void Modelo::anadirVista ( Vista *  vista  ) 

Añade una vista al modelo. Si la vista ya estaba, no la añade. Además, si la vista ya estaba en otro modelo, la saca de ese otro modelo y la apunta a éste.

Parámetros:
vista Vista que se va a añadir.
void Modelo::quitarVista ( Vista *  vista  ) 

Quita una vista del modelo y avisa a la vista de que ya no está en este modelo (para mantener la coherencia).

Parámetros:
vista Vista que se va a quitar.
void Modelo::refrescarVistas ( Vista *  vista = NULL  ) 

Realiza un barrido en todas las vistas forzando el refresco.

Parámetros:
vista Si se especifica este valor, esta vista no se refrescará. Por motivos de eficiencia se asume que ésta es la vista que pidió el refresco y, por tanto, no tiene que refrescarse.
